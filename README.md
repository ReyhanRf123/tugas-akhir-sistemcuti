# Sistem Pengajuan Cuti Karyawan (Tugas Akhir RPLL)

Aplikasi berbasis web untuk manajemen pengajuan cuti karyawan.

## Teknologi
- **Backend:** Java Spring Boot 3
- **Database:** MySQL
- **Frontend:** HTML5 + JavaScript (Fetch API)
- **Deployment:** Railway Cloud

## Fitur
1. Generate data dummy karyawan otomatis.
2. Cek sisa cuti karyawan.
3. Validasi pengajuan cuti (BPM Logic).
4. REST API Endpoint.
